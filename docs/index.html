📖 User Manual

🚀 Getting Started

  • Install the Schedly extension from Chrome Web Store
  • Grant permissions for Gmail and Google Calendar access
  • Configure your preferred AI extraction mode in settings
  • Start scanning your emails for events

⚙️ Configuration

  • Set confidence threshold for event detection (recommended: 0.6-0.8)
  • Choose your default calendar for new events
  • Configure Together AI API key for Hybrid mode
  • Enable/disable automatic event detection

📧 Email Scanning

  • Open any email in Gmail that contains event information
  • Click the Schedly extension icon in your browser toolbar
  • Click "Scan Current Email" to process the content
  • Review detected events with confidence scores

📅 Event Management

  • Preview events before adding to calendar
  • Edit event details like title, time, and location
  • Delete unwanted events to prevent future detection
  • View processing statistics and recent events

🎯 Event Types

  • Academic: Assignments, exams, lectures, lab sessions
  • Professional: Meetings, calls, appointments, interviews
  • Social: Lunch, dinner, coffee, parties
  • Deadlines: Project submissions, assignment due dates

🛠️ Troubleshooting

  • Ensure you're on Gmail (mail.google.com) when scanning
  • Check Chrome permissions for the extension
  • Verify Google Calendar API access is granted
  • Try different AI modes if extraction isn't working
